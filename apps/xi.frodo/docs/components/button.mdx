# Button

**[Figma](https://www.figma.com/file/Kj2qDdyHNRhnJXbzz0kkJV/%F0%9F%96%8D%EF%B8%8F-Components?node-id=21-1631&t=pURu06dOC73EDQ36-4)** • **[Code](https://github.com/xi-effect/xi.monorepo/tree/main/packages/pkg.inputs.button)**

Кнопки позволяют пользователям выполнять действия и делать выбор одним нажатием.

## Базовая кнопка

```jsx live
function BaseButton(props) {
  const handleClick = () => {
    console.log('alert');
  };

  return (
    <>
      <Button color="primary" onClick={handleClick}>
        Click me
      </Button>
    </>
  );
}
```
